# BUSQUEDA-DE-GRAFOS-
Descripción del proyecto
Este proyecto implementa los algoritmos de búsqueda BFS (Búsqueda en Amplitud) y DFS (Búsqueda en Profundidad) utilizando el lenguaje de programación Python.
El programa permite recorrer un grafo representado mediante una lista de adyacencia, mostrando paso a paso cómo se visitan los nodos durante la ejecución de cada algoritmo.
Además, se incluye una visualización gráfica del grafo utilizando las librerías NetworkX y Matplotlib, lo que permite observar de manera más clara la estructura del grafo y las conexiones entre los nodos.


OBJETIVOS
Implementar el algoritmo BFS (Búsqueda en Amplitud).
Implementar el algoritmo DFS (Búsqueda en Profundidad).
Mostrar el recorrido de los nodos paso a paso.
Calcular la memoria utilizada por cada algoritmo.
Visualizar gráficamente la estructura del grafo.

ALGORITMOS IMPLEMENTADOS
🔹 BFS
El algoritmo BFS recorre el grafo por niveles, visitando primero los nodos más cercanos al nodo inicial.

Características:
Utiliza una cola (queue).
Explora todos los vecinos antes de avanzar a otro nivel.
Es útil para encontrar caminos más cortos en grafos no ponderados.

🔹 DFS 
El algoritmo DFS recorre el grafo en profundidad, explorando cada rama antes de retroceder.
Características:
Utiliza recursividad.
Explora completamente un camino antes de visitar otro.
Es útil para análisis de estructuras de grafos.


LIBRERIAS UTILIZADAS

El programa utiliza las siguientes librerías de Python:
collections → manejo de colas (deque)
sys → cálculo de memoria utilizada
time → simulación del recorrido paso a paso
networkx → creación y manejo del grafo
matplotlib → visualización gráfica del grafo

INSTALACION DE DEPENDENCIA 

Antes de ejecutar el programa es necesario instalar las siguientes librerías:
pip install networkx matplotlib

EJECUCION DEL PROGRAMA 

Para ejecutar el programa se debe utilizar el siguiente comando:
python grafo.py
